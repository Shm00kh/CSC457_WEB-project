## 💡 Project Idea

The main goal of this website is to help users:

- Explore the administrative regions of Saudi Arabia  
- Learn basic and clear information about each region  
- Discover important landmarks and attractions  

---

## 🧩 Pages Overview

### • Home Page
- A clean landing page with a background image  
- Contains a central introduction and a **"Start Exploring"** button  
- Includes three sections explaining the purpose of the website  

### • Regions Page
- Displays regions in the form of interactive cards  
- Each card includes:
  - Image  
  - Region name  
  - Short description  
- Users can click on the card or the button to view details  

### • Details Page
- Provides more information about the selected region  
- Includes:
  - Description  
  - General information  
  - Main landmarks  
  - Image gallery
